# FEATURES
- Object oriented language with inheritance and dynamic dispatch
- Dynamic type system
- Exception handling
- Growing arrays (like C++ vectors)
- Tail call optimization (automatically detects RET after CALL and reuses stack frame)
- Tagged integers and chars
- Inline cache
- Cheney's copying GC
